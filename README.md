# 핀테크 개발자 양성과정 학습 기록

핀테크 교육과정에서 진행한 실습 코드와 학습 내용을 정리하는 저장소입니다.

## 교육과정

| 구분   | 학습 내용                   | 관련 폴더    |
| ------ | --------------------------- | ------------ |
| 강의 1 | 핀테크 개요 및 최신 기술    | 이론 수업    |
| 강의 2 | 웹 퍼블리싱 및 UI 개발 기초 | `web/`       |
| 강의 3 | 핀테크 개발 기초(Java)      | `javaPro/`   |
| 강의 4 | 취업 특강                   | 이론 수업    |
| 강의 5 | 데이터베이스 프로그래밍     | `postgredb/` |

## 폴더 구성

- `javaPro/`: Java 문법, 객체지향 프로그래밍, JDBC 실습
- `postgredb/`: PostgreSQL 및 SQL 실습
- `web/`: HTML, CSS, JavaScript 및 반응형 UI 실습
